Tutorial: a Surprised Create

This demo learns a set of predictions. The iRobot Create beeps when the variance of the TD error is bigger than usual.

  1. Requirements: make sure Zephyr plugins are installed in Eclipse (see Download in Zephyr), make sure Eclipse plugins are installed in Eclipse (see Download in RLPark)
  2. Download the project rlpark.example.surprise.tar.gz
  3. Import the project in Eclipse by going to: File>Import... then Existing Project into Workspace then select the project rlpark.example.surprise.tar.gz and follow Eclipse instructions
  4. Follow the instructions on the Zephyr tutorial about Using Zephyr Plugins, starting from Running Zephyr with your client. Note that Autostart is false by default. So, you need to add the id rlpark.example.surprise.createsurprise to the list of program arguments.
  5. Once the example runs, check the code starting from the main class: CreateSurprise.java
